1. 인터페이스
사전적 의미: 두 장치를 연결하는 접속기

인터페이스는 다형성 구현에 주요 기술로 이용된다.
상속을 이용해서 다형성을 구현할 수 도 있지만 실무에서는 인터페이스로 구현하는 경우가 대부분이다.

3.1 public 상수 필드
인터페이스는 public static final 특성을 갖는 상수필드를 가질 수 있다.
인터페이스의 필드는 public static final 갖는다.
상수명(한번 정하면 변할 수 없는 값)
ex) public static final int MAX_VALUE = 10;  // = int MAX_VALUE = 10;

3.2 추상메서드 abstract method
인터페이스는 구현 클래스가 재정의해야 하는 public 추상 메서드를 멤버로 가질 수 있다.
리턴 타입, 메소드명, 매개변수만 기술되고 중괄호{}를 붙이지 않는 메소드이다.
(public abstract) returnType 메서드명(매개변수들...);
